# §0 协作铁律（最高优先 · 不可违反）

- **必须中文沟通**：与用户对话、需求澄清、方案说明、评审回复、PR 描述 — **一律使用中文**；代码标识符 / API 路径 / 配置键仍用英文
- **Git 提交说明必须中文**：`git commit` 的 subject 与 body **必须用中文**写清「为什么改」；type 前缀保留英文惯例：`<type>: <中文 subject，≤50 字>`
  - ✅ `feat: 添加 Pexels 视频源配置说明`
  - ✅ `fix: 修复 WebUI 启动时编码错误`
  - ❌ `feat: add pexels config docs`（英文 subject 禁止）
- **代码注释**：延续「中文注释，英文代码」

本文件为 §0 唯一细则正文（`CLAUDE.md` / `.cursor/rules/` 仅指针）。
